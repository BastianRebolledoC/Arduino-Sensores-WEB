Acerca del proyecto

El objetivo que busca este proyecto es el de crear una aplicación que cuente con una parte de frontend y otra de backend, la primera la cual deberá mostrar la información traída desde un servidor remoto, presentándola en unos gráficos sencillos, resumidos y agradables a la vista del usuario, el cual podrá ver más detalles de estos datos si es que lo desea. Por parte del backend este cuenta con varias capas. El conjunto de estas implementaciones cumplen con el propósito de enseñar y dar a entender cómo funciona una aplicación que tenga varias capas de procesamiento de información, las cuales van desde un arduino captando datos con unos sensores, hasta un gráfico presentado en una página web.
